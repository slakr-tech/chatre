<h1>CHATRE</h1>

Install Dependencies:
```bash
pip install -r requirements.txt
```
\
Run App:
```bash
python __init__.py
```
\
To change the host, change the following variable in settings.py:
```python
self.host = '0.0.0.0'
```
\
To turn off debug mode, change the following variable in settings.py
```python
self.debug = False
```
